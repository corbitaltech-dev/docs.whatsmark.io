The **Media Library** is your workspace's shared file storage — upload once, then reuse the same file anywhere WhatsMark.io asks for media: the chat composer, templates, and flows.

<Warning>
Storage limits depend on your subscription plan. Check your current plan for the available storage capacity.
</Warning>

## Uploading files

<BrowserFrame
  src="/media/img/upload-media.png"
  alt="Upload files to the Media Library"
  caption="Uploading files to the Media Library"
/>

Upload images, video, audio, or documents directly to the library. Once uploaded, a file is available everywhere a media picker appears — you don't need to re-upload the same asset for every template or flow.

## Organizing with folders

- **Create**, **rename**, or **delete** folders to keep files organized.
- **Move** individual files between folders, or **bulk-move** several at once.
- **Bulk-delete** files you no longer need.

## Storage usage

<BrowserFrame
  src="/media/img/storage-use.png"
  alt="Storage usage in the Media Library"
  caption="Storage usage overview"
/>

In the top right corner of the Media Library, you can see your current storage usage:

- **Storage ratio** — displayed as a fraction (e.g., **4.50 MB / 500 MB**)
- **Used storage** — the left number shows how much you're currently using
- **Plan quota** — the right number shows your plan's total storage limit
- **Remaining storage** — see exactly how much storage you have left before reaching your plan limit


## Using the picker

Anywhere WhatsMark.io needs a file — the [Live Chat](/messaging/live-chat) composer, [WhatsApp Templates](/messaging/whatsapp-templates), or [WhatsApp Flows](/automation/whatsapp-flows) — you'll see a **media picker** that opens the same library, so you're always choosing from files you've already organized rather than uploading blind each time.
