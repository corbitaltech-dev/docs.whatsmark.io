The **Media Library** is your workspace's shared file storage — upload once, then reuse the same file anywhere WhatsMark asks for media: the chat composer, templates, and flows.

<Warning>
Storage is limited by your plan. Free-plan workspaces get 50 MB; Professional plans get 500 MB; Reseller plans get 5,000 MB (5 GB).
</Warning>

## Uploading files

Upload images, video, audio, or documents directly to the library. Once uploaded, a file is available everywhere a media picker appears — you don't need to re-upload the same asset for every template or flow.

## Organizing with folders

- **Create**, **rename**, or **delete** folders to keep files organized.
- **Move** individual files between folders, or **bulk-move** several at once.
- **Bulk-delete** files you no longer need.

## Storage usage

The library shows your current storage usage against your plan's quota, so you can see at a glance how much room you have left before you need to clean up or upgrade.

## Using the picker

Anywhere WhatsMark needs a file — the [Live Chat](/messaging/live-chat) composer, [WhatsApp Templates](/messaging/whatsapp-templates), or [WhatsApp Flows](/automation/whatsapp-flows) — you'll see a **media picker** that opens the same library, so you're always choosing from files you've already organized rather than uploading blind each time.
