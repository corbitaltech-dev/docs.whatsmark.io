A **label** is a colored tag that helps you organize and identify contacts. For example, you can create labels like **VIP**, **Hot Lead**, or **Follow Up**. Unlike a status, a contact can have multiple labels at the same time.

WhatsMark.io includes five default labels: **VIP**, **Hot Lead**, **Follow Up**, **Wholesale**, and **Returning Customer**. You can rename them, change their colors, or create your own labels to match your business.

## Where labels show up

- **Contacts page** — each contact shows their labels as colored badges.
- **Contact details** — labels appear in the contact's details panel.
- **Bulk actions** — select many contacts at once and add a label to all of them in one go. See [Bulk Actions & Merge](/contacts/bulk-actions-and-merge).
- **New contacts from messages** — when a message from a new person creates a contact, you can have labels applied for you. Set them in [Contact Configuration](/settings/contact-configuration).

## Creating a label

![New Label form showing fields for name, color, and sort order](/core-setup/img/label-form.png)

<Steps>
<Step title="Open Labels">
  Click **Setup** in the sidebar, then **Labels**.
</Step>

<Step title="Click Create Label">
  The button sits at the top right of the page. A form opens on the right side of the screen.
</Step>

<Step title="Fill in the details">
  - **Name** — a clear tag name, such as **VIP** or **Hot Lead**. Each name can be used only once.
  - **Color** — the badge color shown wherever the label appears. Click the swatch to pick one, or type a color code. Leave it empty and WhatsMark.io picks a color for you.
  - **Sort Order** — the position of this label in lists. It's pre-filled with the next spot, so you can usually leave it as is.
</Step>

<Step title="Save">
  Click **Save**. The label is ready to attach to contacts right away.
</Step>
</Steps>

<Tip>
You can also create a label without leaving the contact form — click the `+` icon beside the **Labels** field while adding or editing a contact.
</Tip>

## Managing labels

<BrowserFrame
  src="/core-setup/img/labels-table.png"
  url="whatsmark.io/labels"
  alt="Labels table listing VIP, Hot Lead, Follow Up, Wholesale, and Returning Customer with color dots, order numbers, and a row actions menu"
  caption="Drag rows to reorder your labels — the same order is used everywhere labels are listed."
/>

From the Labels page you can:

- **Reorder** — drag rows up or down. The same order is used everywhere labels are listed.
- **Edit** — click the menu icon at the end of a row and click **Edit** to change the name, color, or order.
- **Delete** — click the menu icon and click **Delete**.

<Note>
A label that's still in use can't be deleted. If any contacts have it, or it's chosen as a default in [Contact Configuration](/settings/contact-configuration), WhatsMark.io tells you where it's used — clear it there first, then delete.
</Note>
