Statuses represent where a contact is in your lifecycle. For example, "New", "Contacted", "Customer", "Inactive", etc.

## Creating statuses

<Steps>
<Step title="Go to Setup → Statuses">
  Navigate to the statuses section.
</Step>

<Step title="Click Add Status">
  Enter a name for your status (e.g. "Lead", "Opportunity", "Closed").
</Step>

<Step title="Mark as default (optional)">
  If you want this status assigned automatically to newly created contacts, toggle it as default.
</Step>

<Step title="Save">
  Your status is now available.
</Step>
</Steps>

## Managing statuses

- **Edit** a status name
- **Delete** a status — contacts keep their status data but can't be assigned this status going forward
- **Reorder** by dragging — the order here is the order they appear throughout the product
- **Set default**: Only one status can be the default. New contacts are automatically assigned this status.
