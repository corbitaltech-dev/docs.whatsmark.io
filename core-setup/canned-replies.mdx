Canned Replies are reusable text and media snippets that agents can drop into chat with a `/` shortcut. Perfect for common responses like greetings, FAQs, or company information.

## Creating canned replies

Click **Create canned reply** to open the canned reply form:

<BrowserFrame
  src="/core-setup/img/canned-reply-form.png"
  url="whatsmark.io/setup/canned-replies"
  alt="New canned reply form showing fields for title, shortcut, body, attachment, category, sort order, and shared with team"
  caption="Set a title and shortcut, write the reply body, and optionally attach media, a category, and a sort position."
/>

<Steps>
<Step title="Go to Setup → Canned Replies">
  Navigate to the canned replies section.
</Step>

<Step title="Click Create canned reply">
  Open the side sheet to add a new canned reply.
</Step>

<Step title="Fill in the details">
  Configure the canned reply:

  - **Title** — enter a clear name for quick reference, such as **Pricing details**.
  - **Shortcut** — set the `/` shortcut agents type to insert it, such as **pricing**. Letters, numbers, hyphens, and underscores only.
  - **Body** — write the reply text using the formatting toolbar (bold, italic, strikethrough, lists). Required unless you attach media.
  - **Attachment** — optionally drag and drop an image, video, audio, or document to send with the reply.
  - **Category** — optionally group related replies, such as **Sales**, **Support**, or **Billing**.
  - **Sort order** — set the reply position in the table and shortcut list.
  - **Shared with team** — turn this off to keep the reply private to you; on by default so the whole team can use it.
</Step>

<Step title="Save">
  Your canned reply is ready to use in live chat.
</Step>
</Steps>

## Using canned replies in live chat

During a live chat conversation, agents can type `/` in the composer to open the canned reply picker:

<BrowserFrame
  src="/core-setup/img/canned-reply-picker1.png"
  url="whatsmark.io/chat"
  alt="Canned reply picker in the chat composer showing a search field and a matching reply preview"
  caption="Type / to search canned replies by title or shortcut, then select one to insert it into the composer."
/>

- Type `/` to see all available canned replies
- Type `/shortcut` to jump straight to a specific reply
- Search by title or category
- Select a reply to insert its text and attachment into the composer

## Managing canned replies

- **Edit** a reply's title, shortcut, body, attachment, category, sort order, or sharing anytime.
- **Reorder** replies by dragging rows. The same order is used in the shortcut list.
- **Delete** a reply to remove it from the shortcut list.

<Tip>
Each plan has a canned reply limit. The count and remaining quota are shown on the Canned Replies page, and creation is blocked once you reach your plan's limit.
</Tip>
