Canned Replies are reusable text and media snippets that agents can drop into chat with a `/` shortcut. Perfect for common responses like greetings, FAQs, or company information.

## Creating canned replies

<Steps>
<Step title="Go to Setup → Canned Replies">
  Navigate to the canned replies section.
</Step>

<Step title="Click Create Canned Reply">
  Enter a title for quick reference.
</Step>

<Step title="Add your content">
  Type the message text and optionally attach media (images, documents, etc.).
</Step>

<Step title="Create a shortcut (optional)">
  Define a `/` shortcut so agents can quickly insert it. For example, `/welcome` or `/faq`.
</Step>

<Step title="Organize with category (optional)">
  Add a category to group related replies.
</Step>

<Step title="Save">
  Your canned reply is ready to use.
</Step>
</Steps>

## Using canned replies in live chat

During a live chat conversation, agents can:

- Type `/` to see all available canned replies
- Type `/shortcut` to insert a specific reply
- Search by title or category

## Managing canned replies

- **Edit** a reply's content or shortcut anytime
- **Delete** a reply to remove it from the shortcut list
- **Reorder** by dragging to organize them as needed
