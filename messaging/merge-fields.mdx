A merge field is a placeholder that is replaced with a contact's information when the message is sent. For example, if you use Hi `{{contact.first_name}}`, each person will receive their own name, such as Hi Priya or Hi Sam.

## How to add one

In any text box that supports merge fields, type `@`. A list opens, and you can select a field and it add the `{{token}}` into your text box.

You don't need to type it yourself. Just select it from the list.

## Where you can use them

- The [live chat](/messaging/live-chat) composer
- [Canned replies](/core-setup/canned-replies)
- [Campaigns](/messaging/campaigns)
- [WhatsApp template](/messaging/whatsapp-templates) variable values
- [Sending a template to one contact](/contacts/managing-contacts#actions-on-a-single-contact) from a contact row

## Appointment and flow fields

Two places add their own fields to the picker:

- **[Appointments](/ai-and-more/appointments)** — reminder and confirmation messages get appointment fields like the date, time and service.
- **[Automation flows](/automation/flow-builder)** — a flow can use answers the customer gave earlier in the chat.

## When a field is empty

If a contact doesn't have a value for a merge field, that part is stays blank. The message is still sent. You cannot set a default value.
