Messenger Templates are ready-made messages you can save once and reuse anytime on Facebook Messenger. Unlike WhatsApp Templates, Meta approval is not required — you can start using a template immediately after saving it.

<Note>
Before you can create a template, you need to [connect a Facebook Page](/core-setup/channels/messenger) to your workspace.
</Note>

## Creating a template

Click **New Template** to open the template builder.

<BrowserFrame
  src="/messaging/img/messanger/messenger-template-basic-info.png"
  url="whatsmark.io/messaging/messenger/templates/create"
  alt="Basic Info step of the Messenger template builder with fields for Facebook Page, Name, Category, and Template Type, beside a two-step sidebar showing Basic Info and Message"
  caption="Image shows the Basic Info step of the Messenger template builder."
/>

Start with the **Basic Info** step:

- **Facebook Page** – Select the connected Page that the template belongs to.
- **Name** – Enter a template name, up to 100 characters.
- **Category** – Choose a category to organize your templates:
  - **General** – Everyday messages like greetings and follow-ups.
  - **Marketing** – Promotional messages like offers and announcements.
  - **Utility** – Transactional messages like order updates and reminders.
  - **Support** – Customer help messages like FAQs and issue responses.
- **Template Type** – Choose the type of message you want to build.

<Warning>
The template type can't be changed after the template is created.
</Warning>

## Template types

There are five template types, each suited to a different kind of message:

- [**Text**](#text) – A simple text message.
- [**Quick Replies**](#quick-replies) – A message with tappable reply buttons.
- [**Button Template**](#button-template) – A message with action buttons.
- [**Generic Template (Carousel)**](#generic-template-carousel) – Scrollable cards for products or options.
- [**Media**](#media) – An image, video, audio, or file.

### Text

<BrowserFrame
  src="/messaging/img/messanger/messenger-template-text-message.png"
  url="whatsmark.io/messaging/messenger/templates/create"
  alt="Message step of the Messenger template builder with a Message Body field, formatting toolbar, an Add Variable button, and a detected variable with an example value input"
  caption="Image shows the Message step of the Messenger template builder."
/>

A plain text message, up to 2,000 characters. Use the toolbar to format text with **bold**, _italic_, or strikethrough.

To personalize the message, click **+ Add Variable** to add placeholders like `{{1}}` and `{{2}}` (up to 10).

Each variable appears in a list below the body where you type an example value — this only fills the preview. The real values are added when you send the template.

### Quick Replies

<BrowserFrame
  src="/messaging/img/messanger/messenger-template-quick-replies.png"
  url="whatsmark.io/messaging/messenger/templates/create"
  alt="Quick Replies step of the Messenger template builder showing three reply buttons — Get Started, Contact Support, and View Pricing — each with a Button Label and Payload field, and an Add button"
  caption="Image shows the Quick Replies step of the Messenger template builder."
/>

A text message with up to **13** reply buttons — the customer taps one to reply instead of typing.

Each reply button has:

- **Button Label** – The text the customer sees and sends, up to 20 characters.
- **Payload** – Optional. A hidden value your automations can use to know which button was tapped.

### Button Template

<BrowserFrame
  src="/messaging/img/messanger/messenger-template-buttons.png"
  url="whatsmark.io/messaging/messenger/templates/create"
  alt="Buttons step of the Messenger template builder showing two buttons — a URL button labeled Visit Website with a link field, and a Postback button labeled Contact Support with a payload field"
  caption="Image shows the Buttons step of the Messenger template builder."
/>

A text message with up to **3** action buttons. Each button is one of:

- **URL** – Opens a website link.
- **Postback** – Sends an automatic reply in the chat when the customer taps it.

Button titles are limited to 20 characters.

### Generic Template (Carousel)

<BrowserFrame
  src="/messaging/img/messanger/messenger-template-carousel-cards.png"
  url="whatsmark.io/messaging/messenger/templates/create"
  alt="Carousel Cards step of the Messenger template builder showing Card 1 with Title, Subtitle, an Image picker, a Default Tap URL field, and a Buttons section with an Add button"
  caption="Image shows the Carousel Cards step of the Messenger template builder."
/>

A set of 1–10 scrollable cards — ideal for showing products, services, or options. Each card has:

- **Title** – Up to 80 characters.
- **Subtitle** – Optional, up to 80 characters.
- **Image** – Optional. Pick from the [Media Library](/media/media-library) or paste an image link (a wide, landscape image works best).
- **Default Tap URL** – Optional link that opens when the customer taps anywhere on the card.
- **Buttons** – Up to 3 per card (same URL/Postback rules as the [Button Template](#button-template)).

### Media

<BrowserFrame
  src="/messaging/img/messanger/messenger-template-media.png"
  url="whatsmark.io/messaging/messenger/templates/create"
  alt="Media step of the Messenger template builder with a Media Type dropdown set to Image, a selected image preview, a URL field, and an optional Caption field"
  caption="Image shows the Media step of the Messenger template builder."
/>

A single image, video, audio, or file. Pick it from the [Media Library](/media/media-library) or paste a URL, and add an optional caption of up to 2,000 characters.

<Warning>
Every link in a template — button links, card images, and media files — must be a secure link starting with **https://**. Messenger doesn't accept plain http:// links.
</Warning>

## Managing templates

<BrowserFrame
  src="/messaging/img/messanger/messenger-templates-list.png"
  url="whatsmark.io/messaging/messenger/templates"
  alt="Messenger Templates table listing templates with their type, category, Page, and created date, with a row menu open showing Edit and Delete"
  caption="Image shows the Messenger Templates table with a row menu open."
/>

The Messenger Templates table lists all your templates with their type, category, and Page. Use the row menu on any template to:

- **Edit** the template's content. Changes apply to future sends only.
- **Delete** a template.

<Note>
Messenger Templates don't go through any approval process. Every template is ready to use as soon as you save it.
</Note>

## Where you can use them

Once saved, a Messenger template can be used in:

- [**Campaigns**](/messaging/campaigns) – As the content for a Messenger broadcast.
- [**Automations**](/automation/chatbot-flows) – In a flow, using the Messenger Template step.
- [**Live Chat**](/messaging/live-chat) – When starting a new Messenger conversation.
