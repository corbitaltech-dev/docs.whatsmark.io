---
api: "DELETE /api/v1/otp/{reference}"
playground: "simple"
title: "Cancel OTP"
description: "Cancel an active OTP."
---

<Info>
  **Scope required:** `otp:send`
</Info>

Cancel an active OTP. Returns `204`. A verified/expired/cancelled OTP returns `422 otp_inactive`.

<ParamField path="reference" type="string" required>
  Identifier of the reference.
</ParamField>

A successful cancellation returns `204 No Content` with an empty body.

<RequestExample>

```bash Request
curl --request DELETE \
  --url https://api.whatsmark.io/api/v1/otp/9b1f8c2a-4e7d-4a1b-9c3e-2f5a6b7c8d9e \
  --header "Authorization: Bearer YOUR_API_TOKEN"
```

</RequestExample>

<ResponseExample>

```text 204 No Content
```

</ResponseExample>

| error\_code | when |
| --- | --- |
| `unauthenticated` (401) | Missing, invalid or expired API token. |
| `subscription_required` (402) | The workspace has no valid subscription, or a payment is under review. |
| `insufficient_scope` (403) | The token lacks the `otp:send` scope, or the account is suspended/on hold. |
| `otp_inactive` (422) | OTP is no longer active. |
| `quota_exhausted` (429) | The monthly API request quota for the workspace is exhausted. |
