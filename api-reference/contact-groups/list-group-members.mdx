---
api: "GET /api/v1/groups/{group_id}/contacts"
playground: "simple"
title: "List Group Members"
description: "Retrieve the contacts that belong to a group, cursor-paginated."
---

<Info>
  **Scope required:** `groups:read`
</Info>

Cursor-paginated contacts that belong to the group (summary shape).

<ParamField path="group_id" type="string" required>
  Identifier of the group.
</ParamField>

<ResponseField name="data" type="object[]">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Unique identifier of the contact.
    </ResponseField>

    <ResponseField name="display_name" type="string">
      Display name of the contact.
    </ResponseField>

    <ResponseField name="phone" type="string">
      Phone number of the contact.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="links" type="object">
  <Expandable title="properties">
    <ResponseField name="next">
      URL of the next page, or `null`.
    </ResponseField>

    <ResponseField name="prev">
      URL of the previous page, or `null`.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  <Expandable title="properties">
    <ResponseField name="per_page" type="integer">
      Number of results per page.
    </ResponseField>

    <ResponseField name="next_cursor">
      Cursor for the next page, or `null`.
    </ResponseField>

    <ResponseField name="prev_cursor">
      Cursor for the previous page, or `null`.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Request
curl --request GET \
  --url https://api.whatsmark.io/api/v1/groups/1/contacts \
  --header "Authorization: Bearer YOUR_API_TOKEN"
```

### Errors

| Status | error\_code | When |
| --- | --- | --- |
| 401 | `unauthenticated` | Missing, invalid or expired API token. |
| 402 | `subscription_required` | The workspace has no valid subscription, or a payment is under review. |
| 403 | `insufficient_scope` | The token lacks the `groups:read` scope, or the account is suspended/on hold. |
| 429 | `quota_exhausted` | The monthly API request quota for the workspace is exhausted. |
