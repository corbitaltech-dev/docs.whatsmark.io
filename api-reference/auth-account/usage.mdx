<Info>
  **Scope required:** `account:read`
</Info>

Returns current plan limits and consumption for all metered features: contacts, conversations, campaigns, media storage, and API requests. `limit: null` and `unlimited: true` means no cap on that feature.

Common error responses:

| Status | error\_code | When it happens |
| --- | --- | --- |
| 401 | `unauthenticated` | Missing, invalid or expired API token. |
| 402 | `subscription_required` | The workspace has no valid subscription, or a payment is under review. |
| 403 | `insufficient_scope` | The token lacks the required scope, or the account is suspended/on hold. |
| 429 | `quota_exhausted` | The monthly API request quota for the workspace is exhausted. |
