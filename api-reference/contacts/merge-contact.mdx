<Info>
  **Scope required:** `contacts:write`
</Info>

Merge a duplicate contact (`loser_id`) into this one (the path id, the winner). Conversations, identities, campaign recipients, labels/groups and custom fields are re-pointed to the winner; the loser is soft-deleted and marked merged. Returns the winner. Returns `422 invalid_merge` if either side is already merged or you merge a contact into itself.

<ParamField path="contact_id" type="string" required>
  Identifier of the contact.
</ParamField>

<ParamField body="loser_id" type="integer">
  The ID of the duplicate contact to merge into `contact_id` and soft-delete.
</ParamField>

<ResponseField name="data" type="object">
  The merged (winner) contact.

Errors: `401 unauthenticated` for a missing/invalid token, `402 subscription_required` when the workspace has no valid subscription, `403 insufficient_scope` when the token lacks `contacts:write`, `422 invalid_merge` when either contact is already merged or the ids match, and `429 quota_exhausted` when the monthly request quota is exhausted.
