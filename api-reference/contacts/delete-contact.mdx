api: "DELETE /api/v1/contacts/{contact_id}"
playground: "simple"
title: "Delete Contact"
description: "Soft-delete a contact and release one slot of the contacts plan limit."
---

<Info>
  **Scope required:** `contacts:write`
</Info>

Soft-delete a contact. Releases one slot of the `contacts` limit. Returns `204`.

<ParamField path="contact_id" type="string" required>
  Identifier of the contact.
</ParamField>

<RequestExample>

```bash Request
curl --request DELETE \
  --url https://api.whatsmark.io/api/v1/contacts/1 \
  --header "Authorization: Bearer YOUR_API_TOKEN"
```

</RequestExample>

<ResponseExample>

```text 204 No Content
```

</ResponseExample>

Errors: `401 unauthenticated` for a missing/invalid token, `402 subscription_required` when the workspace has no valid subscription, `403 insufficient_scope` when the token lacks `contacts:write`, and `429 quota_exhausted` when the monthly request quota is exhausted.
