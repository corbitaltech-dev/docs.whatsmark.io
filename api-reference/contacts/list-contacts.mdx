---
api: "GET /api/v1/contacts"
playground: "simple"
title: "List Contacts"
description: "Cursor-paginated list of contacts."
---

<Info>
  **Scope required:** `contacts:read`
</Info>

Cursor-paginated list of contacts. Filters: `search`, `type`, `contact_status_id`, `contact_source_id`, `group_id`, `per_page` (5–100, default 50). Merged contacts are excluded.

<ParamField query="per_page" default="50" type="integer">
  Results per page, clamped between 5 and 100.
</ParamField>

<ParamField query="search" type="string">
  Free-text search across contact fields.
</ParamField>

<ResponseField name="data" type="object[]">
  Array of contact objects.
</ResponseField>

<ResponseField name="links" type="object">
  Cursor pagination links.
</ResponseField>

<ResponseField name="meta" type="object">
  Pagination metadata.
</ResponseField>

<RequestExample>

```bash Request
curl --request GET \
  --url https://api.whatsmark.io/api/v1/contacts \
  --header "Authorization: Bearer YOUR_API_TOKEN"
```

</RequestExample>
