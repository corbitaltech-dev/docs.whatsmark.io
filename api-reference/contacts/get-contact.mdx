<Info>
  **Scope required:** `contacts:read`
</Info>

Returns a single contact by ID. Returns `404` if not found or merged.

<ParamField path="contact_id" type="string" required>
  Identifier of the contact.
</ParamField>

```bash Request
curl --request GET \
  --url https://api.whatsmark.io/api/v1/contacts/1 \
  --header "Authorization: Bearer YOUR_API_TOKEN"
```

</RequestExample>

Errors: `401 unauthenticated` for a missing/invalid token, `402 subscription_required` when the workspace has no valid subscription, `403 insufficient_scope` when the token lacks `contacts:read`, `404 not_found` when the contact does not exist or was merged, and `429 quota_exhausted` when the monthly request quota is exhausted.
