---
api: "DELETE /api/v1/statuses/{status_id}"
playground: "simple"
title: "Delete Status"
description: "Deletes a contact status by its identifier."
---

<Info>
  **Scope required:** `statuses:write`
</Info>

Deletes a status. Returns `204`. May return `422` if the status is in use or is the default.

<ParamField path="status_id" type="string" required>
  Identifier of the status.
</ParamField>

<RequestExample>

```bash Request
curl --request DELETE \
  --url https://api.whatsmark.io/api/v1/statuses/1 \
  --header "Authorization: Bearer YOUR_API_TOKEN"
```

</RequestExample>

<ResponseExample>

```json 204 No Content
```

</ResponseExample>

Errors: `401` (missing/invalid token), `402` (no valid subscription), `403` (missing `statuses:write` scope), `422` (status is in use or is the default), `429` (quota exhausted).
