<Info>
  **Scope required:** `templates:read`
</Info>

Length-aware paginated templates, with filters for `channel_id`, `status` (e.g. APPROVED), `category` (e.g. utility), `platform` (e.g. whatsapp), and `per_page`.

<ParamField query="status" type="string">
  Filter templates by status, e.g. `APPROVED`.
</ParamField>

<ParamField query="per_page" type="integer">
  Number of results per page.
</ParamField>
