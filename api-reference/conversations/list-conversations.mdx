<Info>
  **Scope required:** `conversations:read`
</Info>

Cursor-paginated conversations, newest first. Pairs with the `message.received` webhook for a live inbox.

<ParamField query="contact_id" type="integer">
  Filter conversations by contact.
</ParamField>

<ParamField query="channel_id" type="integer">
  Filter conversations by channel.
</ParamField>

<ParamField query="status" type="string">
  Filter by conversation status. One of `open`, `pending`, `assigned`, `resolved`, `closed`.
</ParamField>

<ParamField query="per_page" default="25" type="integer">
  Number of results per page (5–100).
</ParamField>
