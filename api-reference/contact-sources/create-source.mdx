playground: "simple"
title: "Create Source"
description: "Create a source; name and sort_order must be unique."
---

<Info>
  **Scope required:** `sources:write`
</Info>

Create a source. `name` and `sort_order` must be unique.

<ParamField body="name" type="string" required>
  Must be unique within your workspace. Maximum length 100.
</ParamField>

<ParamField body="color" type="string">
  Maximum length 7. Nullable.
</ParamField>

<ParamField body="sort_order" type="integer">
  Must be unique within your workspace. Nullable.
</ParamField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      The source's unique identifier.
    </ResponseField>

    <ResponseField name="name" type="string">
      The source's name.
    </ResponseField>

    <ResponseField name="color" type="string">
      The source's color.
    </ResponseField>

    <ResponseField name="sort_order" type="integer">
      The source's sort order.
    </ResponseField>

    <ResponseField name="is_system" type="boolean">
      Whether this is a system-defined source.
    </ResponseField>

    <ResponseField name="created_at" type="string">
      When the source was created.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Request
curl --request POST \
  --url https://api.whatsmark.io/api/v1/sources \
  --header "Authorization: Bearer YOUR_API_TOKEN" \
  --header "Content-Type: application/json" \
  --data '{
    "name": "Website",
    "color": "#00AA55",
    "sort_order": 1
  }'
```

</RequestExample>

<ResponseExample>

```json 201 Created
{
  "data": {
    "id": 1,
    "name": "Website",
    "color": "#00AA55",
    "sort_order": 1,
    "is_system": false,
    "created_at": "2026-06-26T10:00:00+00:00"
  }
}
```

</ResponseExample>

Errors: `401` missing/invalid/expired token, `402` no valid subscription or payment under review, `403` token missing the `sources:write` scope, `422` validation failed (e.g. `name` required), `429` monthly API request quota exhausted.
