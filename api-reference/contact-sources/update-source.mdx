---
api: "PUT /api/v1/sources/{source_id}"
playground: "simple"
title: "Update Source"
description: "Update a contact source's name, color, or sort order."
---

<Info>
  **Scope required:** `sources:write`
</Info>

Update a contact source.

<ParamField path="source_id" type="string" required>
  Identifier of the source.
</ParamField>

<ParamField body="name" type="string" required>
  Must be unique within your workspace. Maximum length 100.
</ParamField>

<ParamField body="color" type="string">
  Maximum length 7. Nullable.
</ParamField>

<ParamField body="sort_order" type="integer">
  Must be unique within your workspace. Nullable.
</ParamField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      The source's unique identifier.
    </ResponseField>

    <ResponseField name="name" type="string">
      The source's name.
    </ResponseField>

    <ResponseField name="color" type="string">
      The source's color.
    </ResponseField>

    <ResponseField name="sort_order" type="integer">
      The source's sort order.
    </ResponseField>

    <ResponseField name="is_system" type="boolean">
      Whether this is a system-defined source.
    </ResponseField>

    <ResponseField name="created_at" type="string">
      When the source was created.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash Request
curl --request PUT \
  --url https://api.whatsmark.io/api/v1/sources/1 \
  --header "Authorization: Bearer YOUR_API_TOKEN" \
  --header "Content-Type: application/json" \
  --data '{
    "name": "Organic Website",
    "color": "#008844"
  }'
```

</RequestExample>

<ResponseExample>

```json 200 OK
{
  "data": {
    "id": 1,
    "name": "Website",
    "color": "#00AA55",
    "sort_order": 1,
    "is_system": false,
    "created_at": "2026-06-26T10:00:00+00:00"
  }
}
```

</ResponseExample>

Errors: `401` missing/invalid/expired token, `402` no valid subscription or payment under review, `403` token missing the `sources:write` scope, `429` monthly API request quota exhausted.
