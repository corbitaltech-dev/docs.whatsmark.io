## Contact auto-create defaults

When someone messages your WhatsApp number for the first time and no matching contact exists, WhatsMark can create one automatically. Configure how under **Settings → Contacts**:

<Steps>
<Step title="Set global defaults">
  Choose the default **lifecycle stage**, **status**, **source**, **labels**, and **groups** applied to auto-created contacts, plus an **assignment strategy** (e.g. leave unassigned, or assign to a specific team member).
</Step>

<Step title="Override per WhatsApp channel (optional)">
  If you have multiple WhatsApp numbers connected, you can override these defaults for a specific channel — for example, giving contacts from a support number a different default status than contacts from a sales number.
</Step>
</Steps>

<Tip>
Removing a channel-level override falls that channel back to using the global defaults automatically.
</Tip>
