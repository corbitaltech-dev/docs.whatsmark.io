When a new person messages you on WhatsApp or Messenger, WhatsMark.io saves them as a contact automatically. On this page you decide what that new contact gets — their type, status, source, labels, groups, and which team member handles them. Find it under **Settings → Contacts**.

## Global Defaults

These apply to new contacts from all your connected numbers:

- **Type** — The default contact type assigned to new contacts, such as **Guest**, **Lead**, or **Customer**.
- **Status** — The default status assigned to new contacts. If not set, WhatsMark.io uses your default status from the [Statuses](/core-setup/statuses) page.
- **Source** — The default source assigned to new contacts. If not set, WhatsMark.io uses the built-in **WhatsApp** source.
- **Assign To** — Choose how new contacts are assigned:
  - **Unassigned** — New contacts aren't assigned to anyone. Any team member can take over the conversation.
  - **Round Robin** — New contacts are assigned to team members one by one in rotation.
  - **Online Team Member** — New contacts are assigned only to team members who are currently online. If no team members are online, contacts are assigned using all team members.
  - **Custom Team Member** — Every new contact is assigned to the same team member. Select the team member from the **Team Member** field that appears after choosing this option.
- **WhatsApp Channel** — Choose the default WhatsApp number assigned to new contacts.
- **Labels** and **Groups** — optional. assign labels and groups to new contacts automatically.

## Different settings per number

If you have more than one number connected, you can give each number its own settings instead of the global ones:

![Contact Defaults panel for a channel with an Override global defaults switch and fields for lifecycle, status, source, assignment strategy, labels, and groups](/settings/img/channel-contact-defaults.png)

1. Go to your [Channels](/core-setup/channels/whatsapp) page.
2. Click the menu icon on the channel you want and click **Contact defaults**.
3. Turn on the **Override global defaults** switch.
4. Set that channel's own Lifecycle, Status, Source, Assignment Strategy, Labels, and Groups, then save.

From now on, new contacts arriving on that number use these settings — the Global Defaults no longer apply to it. Numbers without an override keep using the Global Defaults.
